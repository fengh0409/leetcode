## ADDED Requirements

### Requirement: Move Zeroes In-Place
系统 MUST 提供原地移动零的数组处理能力，将所有 `0` 移动至数组末尾，同时保持非零元素的相对顺序不变；不得复制数组或创建与输入同规模的额外存储。

- 函数签名：`func MoveZeroes(nums []int)`；原地修改 `nums`，不返回值
- 时间复杂度：`O(n)`；额外空间复杂度：`O(1)`（不计常数级临时变量）
- 稳定性：非零元素的相对顺序 MUST 保持
- 输入范围：`nums` 为 `[]int`，元素可为负、零、正数

#### Scenario: 典型示例（交错零）
- **WHEN** 输入 `[]int{0, 1, 0, 3, 12}`
- **THEN** 输出应为原地变更后的 `[]int{1, 3, 12, 0, 0}`

#### Scenario: 空数组
- **WHEN** 输入 `[]int{}`
- **THEN** 输出应保持为空 `[]int{}`

#### Scenario: 全零数组
- **WHEN** 输入 `[]int{0, 0, 0}`
- **THEN** 输出应保持为 `[]int{0, 0, 0}`（原地操作，无额外复制）

#### Scenario: 无零数组
- **WHEN** 输入 `[]int{1, 2, 3}`
- **THEN** 输出应保持为 `[]int{1, 2, 3}`（相对顺序不变）

#### Scenario: 包含负数与零
- **WHEN** 输入 `[]int{0, -1, 0, -2, 3}`
- **THEN** 输出应为 `[]int{-1, -2, 3, 0, 0}`（稳定，零移至末尾）
